# Nekko Social App

## Description
A cute/kawaii inspired social media app built with JavaScript using the Noroff API.

## Features
- Register & login
- Create, edit, delete posts
- View feed
- View profiles
- Follow/unfollow users
- Search posts

## How to run
1. Clone repo
2. Open index.html or deploy link

## Tech stack
- JavaScript (ES6 modules)
- Noroff API
